## English

*RoltonBasic* - functionaly-based programming language on C++. 

It supporting:
- 2 types
- 4 command
- translating on 1 language

Example of code:

```RoltonBasic
VARINT a 5
VARSTR b "Nagaev"
PRINT "My num is "
PRINT a
PRINT "\nMy colleague is "
PRINT b
TYPE a
TYPE b
```

For building use `build.sh(build.sh)`

## Русский

*RoltonBasic* - функционально-ориентированный язык программирования на C++

Он поддерживает:
- 2 типа
- 4 команды

```RoltonBasic
VARINT a 5

PRINT "My num is "
```
